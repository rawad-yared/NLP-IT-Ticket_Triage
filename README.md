# NLP IT Ticket Triage

End-to-end NLP pipeline for IT ticket routing, urgency prediction, tags, and summary generation.

## Repository Structure

- `notebooks/` Colab notebooks for phased development
- `models/` Saved checkpoints and model download assets
- `app/` Demo application (Streamlit/Gradio)
- `docs/` Project docs, summary, and usage guide
- `data/raw/` Input datasets
- `data/processed/` Cleaned and split datasets
- `outputs/predictions/` Inference outputs
- `outputs/metrics/` Evaluation metrics, confusion matrices, and reports
